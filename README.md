# Learning Management System (LMS)

A full-stack Learning Management System (LMS) built using the MERN stack.  
This platform allows users to browse courses, watch lectures, track progress, and manage subscriptions.

## Features

- User Authentication & Authorization
- JWT-based Secure Login System
- Course Creation & Management
- Upload Course Thumbnails
- Add Video Lectures
- YouTube Video Support
- Progress Tracking
- Subscription Management with Razorpay
- Responsive UI
- Cloudinary Media Management
- Protected Routes
- Admin Dashboard
